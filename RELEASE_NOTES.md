# 묘트 (Myot) 출시 노트

## 버전 1.0.0

### 주요 기능

#### 🐾 펫 건강 관리
- **펫 종류별 맞춤 검사 기준치**: 강아지와 고양이의 검사 항목별 정상 기준치를 자동으로 표시합니다
  - 예: ALP 기준치가 강아지는 20~150, 고양이는 9~53으로 자동 표시
- **건강 차트**: 검사 결과를 시각적으로 확인할 수 있는 차트 기능
  - 일간/주간/월간 뷰 지원
  - 펫 종류에 맞는 기준치 자동 적용

#### 📅 기록 관리
- **기록 탭**: 식사, 활동, 배변, 건강 기록을 카테고리별로 관리
- **달력 기능**: 기록이 있는 날짜에 표시 마커 제공
- **날짜별 기록 조회**: 특정 날짜의 모든 기록을 한눈에 확인

#### 🏥 건강 검사 기록
- **검사 결과 입력**: 다양한 검사 항목의 결과값 입력 및 저장
- **기본 정보 관리**: 병원명, 비용, 체중 등 기본 정보 저장
- **직전 날짜 빠른 이동**: 이전 검사 날짜로 빠르게 이동 가능

#### 🎨 사용자 인터페이스
- **Material 3 디자인**: 현대적이고 직관적인 UI
- **펫 종류별 색상**: 강아지, 고양이 등 펫 종류에 맞는 색상 테마
- **다국어 지원**: 한국어, 영어, 일본어 지원
